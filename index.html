隐私政策

最后更新日期：2026 年 4 月 21 日

药准准（以下简称"本应用"）是一款帮助用户管理日常服药提醒的工具类应用。我们非常重视您的隐私保护，请您在使用本应用前仔细阅读本隐私政策。

核心承诺：本应用的开发者端不收集您的姓名、联系方式、药物内容或任何可识别您身份的信息。仅通过欧盟 GDPR 合规服务 TelemetryDeck 收集极少量匿名事件（如"首次创建药物"）用于改进产品，不与您的身份关联。您的药物与提醒数据默认仅存储在本机；若您选择开启 iCloud 同步（可选），数据将存储于您本人的 Apple iCloud 私有空间，开发者仍无法访问——详见第六章。

一、我们收集哪些信息

本应用不收集任何可识别您身份的个人信息。具体而言：

  • 不要求注册账号
  • 不收集姓名、手机号、邮箱等个人身份信息
  • 不收集位置信息
  • 不收集 IDFA / IDFV 等硬件标识符
  • 开发者服务器不收集您的药物名称、剂量、服药时间或任何健康数据（若您开启 iCloud 同步，数据传输至您本人的 Apple iCloud 账号，开发者同样无法访问——详见第六章）
  • 不使用任何广告 SDK

唯一收集的是匿名事件遥测数据，用于衡量产品活性和改进用户体验，详见下方"七、匿名使用统计"章节。

二、数据存储

您在本应用中录入的药物信息、提醒时间、服药记录等数据，默认存储在您的 iPhone 本地设备上，使用 Apple 提供的 SwiftData 框架进行存储。

本地模式（默认）：

  • 数据不会上传到任何开发者运营的服务器
  • 数据不会与任何第三方共享
  • 卸载本应用后，本机上的数据将被永久删除

iCloud 同步（可选，您可随时开启或关闭）：

  • 数据通过 Apple CloudKit 同步至您本人的 iCloud 私有数据库（Private Database）
  • 仅在您使用同一 Apple ID 登录的苹果设备（iPhone / iPad）之间保持一致
  • 开发者（药准准）无法访问、读取或导出您 iCloud 中的任何数据
  • 数据受 Apple 标准 iCloud 加密保护；若您开启 Apple 高级数据保护（Advanced Data Protection），则为端到端加密
  • 详见下方"六、iCloud 同步（可选）"章节

三、相机权限

本应用可能请求使用您的相机，仅用于拍摄药物照片以便识别。拍摄的照片仅保存在本地（若您开启 iCloud 同步，照片同样仅同步至您本人的 iCloud 账号），不会上传到任何开发者服务器。您可以随时在系统设置中关闭此权限。

四、通知权限

本应用使用 iOS 系统闹钟（AlarmKit）和本地通知来提醒您按时服药。这些提醒完全在您的设备本地处理，不涉及任何网络通信。

五、第三方服务

本应用集成以下第三方服务：

1. TelemetryDeck（匿名产品分析，始终启用）

  • 用途：帮助我们了解哪些功能被使用，改进用户体验
  • 收集内容：详见第七章"匿名使用统计"
  • 提供方：TelemetryDeck GmbH，总部位于德国奥格斯堡
  • 服务器位于欧盟（德国 / 荷兰），受欧盟通用数据保护条例（GDPR）保护
  • 不与您的身份、健康数据或药物信息关联

2. Apple iCloud / CloudKit（云端同步，可选）

  • 用途：在您同一 Apple ID 登录的多台苹果设备之间同步药物与提醒数据
  • 提供方：Apple Inc.
  • 数据存储于您本人的 iCloud 私有数据库（Private Database），由 Apple 标准 iCloud 加密保护
  • 开发者（药准准）无法读取您 iCloud 中的任何数据
  • 受 Apple 隐私政策管辖：https://www.apple.com/legal/privacy/
  • 详见第六章"iCloud 同步（可选）"

本应用不使用：

  • 广告服务
  • 社交登录
  • 第三方云存储（如 AWS / 阿里云 / 腾讯云等）

六、iCloud 同步（可选）

为方便您在多台苹果设备之间保持药物与提醒数据一致，本应用提供可选的 iCloud 同步功能。

启用与关闭：

  • 启用：iOS 系统设置 → Apple ID → iCloud → 药准准 中开启"同步此 iPhone"
  • 关闭：在同一路径下可随时关闭；关闭后本机已录入的数据保留，不会丢失

同步内容：

  • 您录入的药物信息（名称、剂量、备注、药物照片）
  • 闹钟与提醒时间配置
  • 打卡（服药）记录

我们希望您清楚的事实：

  • 同步使用 Apple 的 CloudKit Private Database（私有数据库），数据存储于您本人 Apple ID 下的 iCloud 空间
  • 仅在您同一 Apple ID 登录的苹果设备（iPhone / iPad）之间同步，不对外共享
  • 开发者（药准准）无法访问、读取、修改或导出您 iCloud 中的任何数据——CloudKit Private Database 对应用开发者不可见，这是 Apple 平台的设计
  • 数据受 Apple 标准 iCloud 加密保护；若您在 iOS 系统设置中开启"高级数据保护（Advanced Data Protection）"，则为端到端加密，连 Apple 自身也无法读取
  • iCloud 同步由 Apple 按其隐私政策管理

关于卸载：

  • 卸载本应用会删除本机上的数据
  • iCloud 中已同步的副本不会自动删除——您可通过在新设备（或重装本应用）登录同一 Apple ID 恢复，或在 iOS 系统设置 → Apple ID → iCloud → 管理账户存储 → 药准准 中手动清除

七、匿名使用统计（TelemetryDeck）

为了了解产品是否满足用户需求，我们通过 TelemetryDeck 收集少量匿名使用事件。

我们收集：

  • 每台设备首次安装时生成的随机 UUID（不包含任何个人信息；重装 App 会生成新 UUID）
  • 事件名称（例如"首次创建药物"、"首次打卡"）
  • App 版本号
  • 系统语言
  • 事件触发时间

我们不收集：

  • 您的姓名、邮箱、电话号码、Apple ID
  • 您的药物名称、剂量、服药时间
  • 任何健康数据或医疗信息
  • 您的位置
  • 设备硬件标识符（IDFA / IDFV）

数据存储在 TelemetryDeck 位于德国 / 荷兰的服务器，受欧盟 GDPR 保护。卸载本应用后，所有本地数据与匿名 UUID 一并删除，TelemetryDeck 端的历史事件数据也将无法追溯到您。

八、儿童隐私

本应用不针对 13 岁以下儿童设计，也不会有意收集儿童的个人信息。

九、隐私政策的变更

如果我们对本隐私政策进行修改，将在本页面发布更新后的版本，并更新"最后更新日期"。建议您定期查看本政策。

本次更新（2026 年 4 月 21 日）主要内容：新增可选的 iCloud 同步功能的完整披露；明确区分"本地模式"与"iCloud 同步"两种数据存储路径；在第三方服务中加入 Apple iCloud / CloudKit。
